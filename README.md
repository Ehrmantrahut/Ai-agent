# AI Agent with Tools

A conversational AI agent built with Groq API and Llama 3.3-70b model.

## Features
- Natural language conversation in any language
- Math calculator tool
- Weather lookup tool
- Extensible tool system

## Tech Stack
- Python 3.13
- Groq API (Llama 3.3-70b-versatile)
- LLM-powered responses
